✦ Edit App Tutorial

Add Reels
in Order

Follow these 7 simple steps to add your Instagram Reels in exactly the order you want inside the Edit app.

1

Install / Update the Edit App

Go to the App Store or Play Store, search for Edit and install or update it to the latest version.

2

Open Edit & Create New Project

Launch the Edit app. Tap New Project to start a fresh video project.

3

Tap on "Links"

In the bottom toolbar, tap the Links button (Instagram icon highlighted in yellow).

4

Tap "Reels"

A popup appears with Accounts and Reels — tap Reels to continue.

5

Select "My Reels" & Pick in Order

Tap the My Reels tab. Now tap each reel one by one in the exact order you want them in your video.

6

First Reel Added to Timeline

Your first selected reel appears in the timeline as a prapul.jain clip. Tap more reels to keep adding.

7

All Reels Added in Order ✅

All your reels are now stacked in the timeline in the exact order you tapped them. Done!

🔑

Key Tip

The order you tap = order in the timeline. Tap your first reel first, second reel second — and they'll appear exactly like that!
